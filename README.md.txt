## About Dataset
This dataset contains pizza sales transactions including order date, pizza type, category, size, quantity, and total price.

## Tools Used
- SQLite
- DB Browser for SQLite
- SQL

## Key Queries & Insights

### 1. Total Orders
Calculated total unique orders placed.

### 2. Total Revenue
Measured overall income from pizza sales.

### 3. Best-Selling Pizzas
Identified top 5 pizzas based on quantity sold.

### 4. Revenue by Category & Size
Analyzed customer preferences.

### 5. Average Order Value
Calculated average revenue per order.

## Key Takeaways
- Certain pizza categories generate higher revenue
- Medium and large pizzas sell more
- Few pizzas dominate total sales
